# Лабораторна робота №4. Інтерактивний фільтр сигналів.

Мета: Розробка програми з графічним інтерфейсом для генерації та фільтрації гармонійних сигналів у реальному часі.

Інструкція з використання:
1. Запуск. Виконайте команду python lab4.py.
2. Керування хвилею. Використовуйте ліві слайдери (Amplitude, Frequency, Phase) для налаштування чистої гармоніки.
3. Керування шумом. Праві слайдери (Noise Mean, Noise Cov) регулюють інтенсивність завад.
4. Фільтрація. Слайдер Cutoff змінює частоту зрізу фільтра, що дозволяє очистити сигнал (червона лінія).
5. Чекбокс Show Noise вмикає або вимикає відображення зашумленого сигналу.
6. Кнопка Reset миттєво повертає всі параметри до початкових значень.
