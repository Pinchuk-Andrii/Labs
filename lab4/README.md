# Лабораторна робота №4. Інтерактивний фільтр сигналів.

Мета: Розробка програми з графічним інтерфейсом для генерації та фільтрації гармонійних сигналів у реальному часі.

**Інструкція з налаштування та запуску (VENV)**
1. Створення віртуального середовища:
   ```bash
   python -m venv .venv

**Інструкція з використання інтерфейсу**
1. Після запуску відкриється вікно з інтерактивним графіком, що містить три типи сигналів: чистий (пунктир), зашумлений та відфільтрований.

**Керування слайдерами:**
1. Amplitude, Frequency, Phase: Змінюють параметри основної гармоніки.
2. Noise Mean, Noise Cov: Налаштовують середнє значення та дисперсію випадкового шуму.
3. Cutoff: Регулює частоту зрізу фільтра низьких частот (дозволяє очистити сигнал від шуму).

**Додаткові елементи:**
1. Show Noise: Вмикає або вимикає відображення зашумленого сигналу та результату фільтрації.
2. Reset: Миттєво повертає всі налаштування (слайдери) до початкових значень.
