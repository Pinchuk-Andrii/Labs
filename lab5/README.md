# Лабораторна робота №5
Тема: Наука про дані, обмін результатами та початковий аналіз

## Опис проєкту
Цей додаток призначений для інтерактивного аналізу індексів VCI, TCI та VHI на основі даних, отриманих у лабораторній роботі №2. 

## Функціонал
1. Вибір часового ряду (VCI, TCI, VHI) через випадаючий список.
2. Фільтрація за областю України.
3. Вибір інтервалу тижнів та років за допомогою слайдерів.
4. Три вкладки для перегляду:
    1) Таблиця: відфільтровані дані з можливістю сортування.
    2) Графік: візуалізація часового ряду для обраної області.
    3) Порівняння: графік обраної області на фоні всіх інших областей України.

## Як запустити
1. Встановіть необхідні бібліотеки:
   `pip install -r requirements.txt`
2. Запустіть додаток із кореневої директорії проєкту (Labs) командою:
   `streamlit run lab5/lab5.py`
